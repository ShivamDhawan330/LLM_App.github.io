Efficient Learning System with AI Agents

Summary

Link to the App

Objective : To have an efficient learning assistant that will cut short your time to research / search for a topic. Just decide what you want to learn, create your schedule and thats it. The AI Agent will fetch topic from the list one by one and fetch you content, evaluate you based on the topic you studied and only after your confirmation will move to next topic.

Overview

  1. Create personalized learning schedules with topics of their choice.
  2. Learn topics step-by-step where each topic is explained in simple words along with a real-life example.
  3. Test their understanding using dynamically generated AI-based quizzes (True/False format).
  4. Track their progress across different schedules.
  5. Use AI guidance for further clarification if a topic is still confusing.

Features:

  1. User Authentication : Signup/Login system using username, email, and password.
  2. Customized Schedule: 1) Name your schedule (example: "Introduction to Robotics"). 2) Relatable examples are included to deepen understanding.
  3. AI-Powered Topic Explanation: 1)Each topic is explained by an AI (Gemini model) in beginner-friendly language. 2) RouterQueryEngine: Router routes queries to the correct query engine based on their nature (summary vs. context-specific).
  4. Dynamic Quiz Generation: 1)After explanation, a quick 5-question True/False quiz is generated to test comprehension. 2) User answers are compared with correct answers. 3) Final Score is displayed.
  5. Continue Learning Flow: 1)User moves to next topic if they understood the previous one. 2) If confused, a deeper bullet-pointed explanation is generated.

Tech Stack:

    1. Backend: Flask (Python)
    2. Frontend: HTML, Bootstrap 5, Jinja Templates
    3. Database: PostgreSQL (managed via Supabase)
    4. LLM: Google Gemini-2.0-flash-lite
    5. AI Agent Framework: Langchain

    Future Consideration:

    1. Add Google OAuth login for faster signup.
    2. Display analytics dashboard (e.g., % topics completed, average quiz score).
    3. Email reminders: "Hey, you haven't learned anything today!"
    4. Mobile-optimized UI (currently desktop-friendly).

    Key Motivations : Promote lifelong learning in an engaging way. Enable users to control their own learning pace.
